# web-scraping-projects
In this Repo I make some projects which I have mentioned below 

**How to run:**
first of all You should install BeautifulSoup4
run these command in your terminal
1. pip install requests BeautifulSoup4
2. python3 Project_Name.py

**My Projects:**
1. bs.py : This project scrape all Books at every page from "books.toscrape.com"
2. qoute.py : This project scrape all Quotes at every page from "quotes.toscrape.com"
3. job.py : This project scrape all job details like Job Title, Company Name, Location, Date from "https://realpython.github.io/fake-jobs/"
4. news_scraper_auto.py : This project scrapes entire page of the Technology section of "The Time of India"
